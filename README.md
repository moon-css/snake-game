# 贪吃蛇游戏

一款在浏览器中运行的贪吃蛇小游戏，含多关卡与复古风格界面。

## 本地运行

下面说明如何把项目跑在你自己的电脑上，适合**第一次接触前端项目**的开发者：你只需要会「复制命令、在终端里粘贴执行」即可。

### 你需要先准备什么

- **一台电脑**（Windows / macOS / Linux 均可）。
- **Node.js**：用来运行 JavaScript 工具和本地开发服务器。  
  - 打开 [Node.js 官网](https://nodejs.org/)，下载并安装 **LTS（长期支持版）**，安装时一路「下一步」即可。  
  - 安装完成后，打开终端（Windows 可在文件夹地址栏输入 `cmd` 回车，或搜索「终端」「PowerShell」），输入下面命令，若能看到版本号说明安装成功：

    ```bash
    node -v
    npm -v
    ```

### 拿到项目代码

- 若使用 **Git**：在终端进入你想存放项目的文件夹，执行：

  ```bash
  git clone <仓库地址>
  cd <项目文件夹名>
  ```

- 若从网页 **下载 ZIP**：解压后，记住解压出来的文件夹路径，后面终端要进入这个文件夹。

**项目根目录**指包含 `package.json` 的那一层文件夹，以下所有命令都要在这个目录下执行。

### 安装依赖（只需做一次，或换电脑后再做）

依赖就是项目用到的第三方库（在 `package.json` 里列出）。第一次运行前必须先安装，终端进入项目根目录后执行：

- 若本机已安装 [Yarn](https://classic.yarnpkg.com/lang/en/)（本仓库带有 `yarn.lock`，推荐）：

  ```bash
  yarn install
  ```

- 或使用 npm（Node 自带）：

  ```bash
  npm install
  ```

过程可能持续几分钟，结束后会出现 `node_modules` 文件夹（体积较大，一般不要手动改它；若已加入 `.gitignore` 则不会提交到 Git）。

### 启动开发环境（改代码时常用）

在项目根目录执行：

```bash
yarn dev
```

或：

```bash
npm run dev
```

终端里会打印类似 `Local: http://localhost:5173/` 的地址。用浏览器（Chrome、Edge 等）打开该地址即可看到游戏。  
修改 `src` 里的代码并保存后，页面通常会**自动刷新**，无需反复手动刷新。

要停止服务器：在运行 `dev` 的那个终端窗口按 **Ctrl + C**。

### 打包正式构建（可选）

若要生成可部署的静态文件（例如上传到服务器）：

```bash
yarn build
```

或 `npm run build`。完成后会在项目下生成 **`build`** 文件夹，里面是打包好的网页资源。

### 常见问题

| 现象 | 可尝试的处理 |
|------|----------------|
| 提示找不到 `node` 或 `npm` | 重新安装 Node.js，并**重新打开**终端后再试。 |
| 提示找不到 `vite` 或某模块 | 删除 `node_modules` 后重新执行 `yarn install` 或 `npm install`。 |
| 端口被占用（5173 打不开） | 关闭占用端口的程序，或看终端是否提示了其它端口号，用新地址打开。 |
| `yarn` 命令不存在 | 可用 `npm install` 代替安装依赖，用 `npm run dev` 代替 `yarn dev`。 |

若仍无法运行，把终端里的**完整报错文字**复制下来搜索或向他人求助，会更容易定位问题。

## 在线游玩

在浏览器中打开：

**<https://moon-css.github.io/snake-game/>**

首次加载会预载图片与音频，请稍候；首页进度条走完后即可点击开始。

## 游戏说明

| 操作 | 说明 |
|------|------|
| **方向键** 或 **W / A / S / D** | 控制蛇的移动方向 |
| **空格** | 游戏中暂停 / 继续 |
| **开始** | 从首页进入关卡选择，再选择要玩的关卡 |
| **返回 / 关闭** | 在选关页或游戏中按界面按钮返回 |

部分关卡有特殊规则（例如影子、红绿灯、成长地图等），可在游戏中自行体验。
